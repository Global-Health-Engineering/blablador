# Blablador

A `python` package for API access to [https://helmholtz-blablador.fz-juelich.de/](https://helmholtz-blablador.fz-juelich.de/).

## Installation

**Requirements:**
``` bash
pip install -e .
```

**Usage design:**
``` bash
git submodule add https://github.com/Global-Health-Engineering/blablador
```
